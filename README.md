# OrthanFlow

## Welcome to OrthanFlow ! 

After cloning the repository : 

## To install the backend

``` bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows : venv\Scripts\activate
pip install -r requirements.txt
```

## To install the frontend 
``` bash
cd frontend/frontv2
npm install
```

## To run the backend 
``` bash
python run.py
```

## To run the frontend 
``` bash
npm run dev
```
